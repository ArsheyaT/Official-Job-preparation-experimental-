Student Portfolio Project

This portfolio project's layout is contemporary and interactive, showcasing interesting content areas. In order to generate hierarchy and visual interest, the structure places a strong emphasis on carefully crafted headings that direct the spectator through various parts. Each heading is stylized with unique typeface and space. The website incorporates hover buttons, which offer seamless transitions and subtle effects that encourage interaction while preserving a polished appearance. Additionally, the project includes photos that have dropdown links integrated into them, enabling users to browse to external resources or explore relevant topics straight from visual elements.

Web Design Project

Using HTML, Flexbox, and media queries for responsiveness, this business website design project focuses on a simple dashboard with deliberate color contrast and hover states. Flexbox is used to organize the desktop layout, which consists of a primary content area that grows dynamically and a horizontal navigation bar at the top. Buttons have hover effects that add subtle shadows and change the backdrop color, and the icons inside of them scale significantly to give tactile input.

Official Birthday email project

An autoplay, muted, looping hero video in a responsive HTML birthday email with strong fallbacks for compatibility with all email clients. Accessibility and smooth degradation are given top priority in the design; in the event that video is not supported, recipients are shown with a high-quality poster image or animated GIF, maintaining the festive atmosphere and the clarity of the call to action.
